Core Technology

Dr Onco core technology is about diagnosing lung cancer from the lung X-Ray images of patients using self laid convolutional neural network and imaging process.

The model we featured in, consists of 3 layers which has been trained on 1560 patients (approx 6000 images).

The model takes a chest X-ray image as input and outputs the probability of lung nodules in patient's report.
